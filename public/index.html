Fachetti Back-End

Rutas

Categories

    GET
  • '/': active categories
  • '/banned': active and banned categories

    • POST
  • '/': new category. Auth required.

    • PUT
  • '/:id': update category by id. Auth required.
  • '/banned/:id': logic deleted for category. Auth required.

    • DELETE
  • '/:id': delete category by id. Auth required.

Products

    GET
  • '/': active products
  • '/banned': active and banned products
  • '/:id': get one product by id
  • '/categories/:category': get all products by specific category name
  • '/categories/:category/banned': get all products by specific category name (active and banned)

    • POST
  • '/': new product. Auth required.

    • PUT
  • '/:id': update publication by id. Auth required.
  • '/banned/:id': logic deleted for publication. Auth required.

    • DELETE
  • '/:id': delete publication by id. Auth required.

Publications

    GET
  • '/': active publications
  • '/banned': active and banned publications
  • '/:id': get one publications by id
  • '/important': get all important publications

    • POST
  • '/': new publication. Auth required.

    • PUT
  • '/:id': update publication by id. Auth required.
  • '/banned/:id': logic deleted for publication. Auth required.

    • DELETE
  • '/:id': delete publication by id. Auth required.

Questions

    GET
  • '/': active questions. Answered, readed, page and sizePage by query.
  • '/:id': get one question by id

    • POST
  • '/': new question. Auth required.

    • PUT
  • '/:id': update question by id. Answered and readed by query. Auth required.

    • DELETE
  • '/:id': delete question by id. Auth required.
